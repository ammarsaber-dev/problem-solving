# 1295 - Find Numbers with Even Number of Digits

**Tags:** 

## Problem

Given an array `nums` of integers, return how many of them contain an even number of digits.

## Examples

| Input | Output |
|-------|--------|
| `nums = [12,345,2,6,7896]` | `2` |
| `nums = [555,901,482,1771]` | `1` |

## Constraints

- `1 <= nums.length <= 500`
- `1 <= nums[i] <= 10^5`
