# 050A - Domino Piling

Tags:

## Problem

You are given a `2 x n` rectangle. You need to maximize the number of `2 x 1` dominoes placed inside the rectangle (they can overlap).

Input: Two integers m and n (1 ≤ m, n ≤ 1000)

Output: Maximum number of dominoes.
